Pages and Views

This article explains the basics of altering pages and views in Microsoft Word.

View tab

Under the View tab you have several options to zoom and have choose where one or more pages are shown on screen.


Zoom keyboard shortcuts:
  • Ctrl and + to zoom in / Ctrl and - to zoom out
  • If you're using a mouse a much faster way to zoom is to hold down Ctrl + mouse wheel up to zoom in and Ctrl + mouse wheel down to zoom out. Some laptop mousepads offer this function down one side as well.
  • Ctrl + 0 (zero) will reset zoom level to 100%
