Formatting text

On the Home tab of the ribbon are formatting tools. This article explains what each does.


Clipboard tools

Format Painter

This is a useful tool which applies formatting from selected formatted text to other text. This is much faster to apply the same custom formatting elsewhere.

Note: For headings you should use Styles (see further below), the format painter can then be used to apply a heading's heading style onto other text.
